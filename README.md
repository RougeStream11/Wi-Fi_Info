### Wi-Fi Info Android Project

***Shows information about network you are connected to.***

* __SSID__
* __BSSID__
* __IP__
* __RSSI__
* __Frequency__
* __Network Speed__
* __Network ID__
* __MAC Address__
* __WPA Supplicant State__

Go to **Releases** to get the latest app version! **(1.3_b)**
